# Contribuindo para o g-clima 🌤

Obrigado por seu interesse em contribuir para o g-clima! Este é um projeto open-source e adoraríamos receber sua ajuda.

## Como contribuir

1.  **Reportar Bugs**: Abra uma *Issue* detalhando o erro.
2.  **Sugerir Melhorias**: Quer uma métrica nova? Abra uma *Issue* de sugestão.
3.  **Enviar Código**:
    *   Faça um Fork do projeto.
    *   Crie uma branch para sua feature (`git checkout -b feature/nova-feature`).
    *   Faça o commit das suas alterações.
    *   Envie para o seu fork (`git push origin feature/nova-feature`).
    *   Abra um **Pull Request**.

## Padrões de Código

*   Use Type Hints.
*   Documente novas funções.
*   Se possível, adicione testes em `/tests`.
